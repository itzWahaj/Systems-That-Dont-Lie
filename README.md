# Storytelling Portfolio

A narrative-driven portfolio website built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- **Cinematic Storytelling**: Chapter-based scrolling journey.
- **MDX Project Pages**: Data-driven project content.
- **SEO Optimized**: Dynamic sitemap, robots.txt, and Open Graph tags.
- **Performance**: Optimized fonts and minimal client-side JS.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  **Open [http://localhost:3000](http://localhost:3000)** with your browser.

## Project Structure
- `src/app`: App Router pages and layouts.
- `src/components`: Reusable UI components.
- `src/content`: MDX files for projects.
- `src/lib`: Utility functions.

## Adding Projects
Add a new `.mdx` file to `src/content/projects/` with the required frontmatter.
